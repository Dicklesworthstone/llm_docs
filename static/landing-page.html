A groupby operation involves some combination of splitting the object, applying a function, and combining the results. This can be used to group large amounts of data and compute operations on these groups. Parameters ---------- by : mapping, function, label, or list of labels Used to determine the groups for the groupby. If by is a function, it's called on each value of the object's index. If a dict or Series is passed, the Series or dict VALUES will be used to determine the groups (the Series' values are first aligned; see .align() method). If a list or ndarray of length equal to the selected axis is passed, the values are used as-is to determine the groups. A label or list of labels may be passed to group by the columns in self. Notice that a tuple is interpreted as a (single) key. axis : {0 or 'index', 1 or 'columns'}, default 0 Split along rows (0) or columns (1). level : int, level name, or sequence of such, default None If the axis is a MultiIndex (hierarchical), group by a particular level or levels. as_index : bool, default True For aggregated output, return object with group labels as the index. Only relevant for DataFrame input. as_index=False is effectively "SQL-style" grouped output. sort : bool, default True Sort group keys. Get better performance by turning this off. Note this does not influence the order of observations within each group. Groupby preserves the order of rows within each group. group_keys : bool, default True When calling apply, add group keys to index to identify pieces. observed : bool, default False This only applies if any of the groupers are Categoricals. If True: only show observed values for categorical groupers. If False: show all values for categorical groupers. dropna : bool, default True If True, and if group keys contain NA values, NA values together with row/column will be dropped. If False, NA values will also be treated as the key in groups. Returns ------- DataFrameGroupBy Returns a groupby object that contains information about the groups. See Also -------- resample : Convenience method for frequency conversion and resampling of time series. Notes ----- For more information on the groupby, see the [user guide](https://pandas.pydata.org/pandas-docs/stable/groupby.html). # DataFrame.groupby ```python DataFrame.groupby(by=None, axis=0, level=None, as_index=True, sort=True, group_keys=True, observed=False, dropna=True) -> 'DataFrameGroupBy' ``` Groups DataFrame by specified keys and returns a GroupBy object for aggregation operations. ## Parameters - `by`: Determines grouping criteria. Can be: - Column name or list of names - Function applied to index - Dictionary or Series (values determine groups) - Array of same length as axis - `axis`: `0`/`'index'` (default) or `1`/`'columns'` - split direction - `level`: Group by MultiIndex level(s) - `as_index`: If `True` (default), groups become index in result - `sort`: If `True` (default), sort group keys - `group_keys`: If `True` (default), add group keys to index when applying functions - `observed`: For categorical groupers, show only observed values (`True`) or all values (`False`, default) - `dropna`: If `True` (default), drop NaN values in group keys ## Returns - `DataFrameGroupBy`: Object with group information for aggregation ## Key Operations on GroupBy Objects After grouping, common operations: - `.sum()`, `.mean()`, `.max()`, `.min()` - aggregate functions - `.agg([func1, func2])` - apply multiple aggregations - `.transform(func)` - apply function to each group, return same-sized object - `.filter(func)` - filter groups based on condition - `.apply(func)` - apply arbitrary function to each group ## Examples Basic grouping: ```python df.groupby('A').sum() # Group by column 'A' and sum values ``` Multiple columns (hierarchical index): ```python df.groupby(['A', 'B']).sum() ``` Custom grouping function: ```python df.groupby(len).sum() # Group by length of index values ``` SQL-style output (no index): ```python df.groupby('A', as_index=False).sum() ``` Grouping by columns instead of rows: ```python df.groupby(['A'], axis=1).sum() ```
